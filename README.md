#ptech-board
### - Spring boot를 사용한 토이프로젝트


* Environment
    * Java 1.8
    * SprinBoot 2.5.1
    * Spring-Data-JPA
    * Spring Security
  
<br>

* 개발환경
    * Intellij - Tyhmeleaf live reload

<br>

* 개발환경 구동
    * [Install sass](https://sass-lang.com/install)
    ```sh
    # Homebrew
    brew install sass/sass/sass
    ```
    * Run application
    * run sass watch
    ```sh
    ./run-sass-watch.sh
    ```
